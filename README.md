# Artificial-Intelligence

## Algorithms
* BFS (Breadth-first search)
* DFS (Depth-first search)
* A-Star
* Best-first search
* Hill-Climbing

## Heuristics
* Manhattan distance
* Euclidean distance
* Minkowski distance, with p = 3
* Minkowski distance, with p = 4
* Average between Manhattan distance and Euclidean distance


## Compile
```
make all
```

## Execute
### Example
```
make run input=1.in
```
